# 04_财务数据字典

> 主数据字典基于已发现表头、模板和公式字段建立。字段是否为公司唯一标准字段，除非来源明确，否则均需人工核实。

| 字段名称 | 含义 | 来源 | 使用部门 | 是否核心字段 |
|---|---|---|---|---|
| 客户编号 | 订单/收支/押金/流水关联客户的编号，如 LS、KGK、LN 等前缀 | 收支登记表、销售流水表、定金单、客户账单整理 | 财务、销售、客服、业务 | 是 |
| 客户 | 客户主体或客户归属，部分表用客户编号或客户归属替代 | 定金单、客户账单整理、销售统计表 | 客服、销售、财务 | 是 |
| 群备注 | 服务群或订单群名称，常含日期、车型、客户编号 | 定金单 | 客服、财务、车队 | 是 |
| 用车时间 | 预定/用车日期 | 定金单 | 客服、销售、车队 | 是 |
| 出车时间 | 车辆交付/订单开始时间 | 销售流水表、销售统计表、客服绩效表 | 销售、客服、车队 | 是 |
| 回车时间 | 车辆归还/订单结束时间 | 销售流水表、销售统计表 | 销售、财务、车队 | 是 |
| 车型 | 客户咨询或实际使用车辆型号 | 定金单、销售流水表、结算单 | 客服、销售、车队 | 是 |
| 车牌 | 实际车辆识别字段 | 销售流水表、销售统计表、车队绩效表、维修整备表 | 车队、财务、销售 | 是 |
| 单价/车辆单价/日租成交价 | 订单按天或全包报价的单价字段 | 定金单、销售流水表、销售统计表、通用结算单 | 销售、客服、财务 | 是 |
| 实际订单天数/实际天数/数量（天） | 结算用天数 | 销售流水表、销售统计表、通用结算单 | 财务、销售 | 是 |
| 租金收入 | 销售流水表中的租金收入字段 | 销售流水表 | 销售、财务 | 是 |
| 最终销售金额 | 销售统计表中的订单销售金额，部分表公式为车辆单价 x 实际订单天数 | 员工提成表/销售统计表 | 销售、财务 | 是 |
| 租车成本 | 订单对应外调、挂靠或其他车辆成本 | 员工提成表/销售统计表 | 财务、销售 | 是 |
| 利润 | 销售统计表字段，部分表公式为最终销售金额 - 租车成本 | 员工提成表/销售统计表 | 财务、销售 | 是 |
| 定金收款金额 | 客户下单时已收定金 | 定金单 | 客服、财务 | 是 |
| 尾款 | 定金后剩余待收款 | 定金单、客服绩效表 | 客服、财务 | 是 |
| 押金 | 车辆或订单押金 | 定金单、收支登记表 | 财务、客服 | 是 |
| 违章押金 | 还车后可能退还或暂扣的违章相关押金 | 收支登记表、绩效违规表 | 财务、车队 | 是 |
| 收款方式 | 客户付款渠道，如收钱吧、微信、现金、账上余额等 | 定金单、收款登记表 | 财务、客服 | 是 |
| 支付方式 | 公司付款渠道 | 支出登记表、日收支明细 | 财务 | 是 |
| 收款类型 | 收款分类，如租金、押金、定金、其他 | 收款登记表、月度收支汇总 | 财务 | 是 |
| 付款类型 | 支出分类，如退客户押金、同行结算、挂靠结算、维修结算、工资社保、其他 | 支出登记表、日收支明细 | 财务 | 是 |
| 金额 | 收付款金额 | 收款登记表、支出登记表、结算单 | 财务 | 是 |
| 摘要/报销摘要 | 收支事项说明 | 收款登记表、支出登记表 | 财务、业务 | 是 |
| 备注 | 补充说明，可能含异常、负责人、客户归属 | 多数表 | 全部门 | 否 |
| 销售/销售人姓名 | 订单归属销售 | 定金单、销售流水表、销售统计表 | 销售、财务 | 是 |
| 客户归属 | 定金单中用于归属客户来源/销售体系的字段 | 定金单 | 客服、销售 | 是 |
| 车主结算 | 与车主或挂靠车相关的付款/结算事项 | 日收支明细、挂靠车辆账单候选 | 财务、业务 | 是 |
| 同行结算 | 外调/同行车辆付款结算 | 日收支明细、同行付款申请 PDF | 财务、车队 | 是 |
| 挂靠结算 | 挂靠车辆付款结算 | 日收支明细、挂靠车辆账单候选 | 财务、业务 | 是 |
| 销售提成 | 销售佣金或提成字段 | 未发现明确统一字段；有员工提成表和销售统计表 | 财务、销售 | 无法确认 |
| 订单状态 | 订单所处状态，如咨询、定金、出车、还车、结算、归档 | 由客服绩效流程可推导出节点，但未发现统一字段 | 客服、财务、车队 | 无法确认 |
| 车辆状态 | 车辆可出车、在外、维修、整备等状态 | 在外车辆表、维修整备表、车辆台账候选 | 车队、财务 | 是 |
| 问题严重程度 | 维修/整备问题等级：严重、一般、轻微 | 车辆维修、整备管理相关表格 | 车队、财务 | 是 |
| 是否影响正常出车 | 维修/整备是否影响车辆出车 | 车辆维修、整备管理相关表格 | 车队、财务 | 是 |
| 必修项/可缓修项 | 维修项目是否必须立即处理 | 车辆维修、整备管理相关表格 | 车队、财务 | 是 |
| 不修风险说明 | 维修不处理的风险说明 | 车辆维修、整备管理相关表格 | 车队、财务 | 是 |
| 审核人 | 维修/整备或其他事项审核人 | 车辆维修、整备管理相关表格 | 管理、车队、财务 | 是 |
| CRM上传状态 | 出车信息是否已上传 CRM | 客服绩效表中要求当日上传，但未发现独立字段 | 客服、销售 | 无法确认 |
| 收款凭证 | 尾款后制作并发送群内的凭证 | 客服绩效表、绩效违规表 | 财务、客服 | 是 |
| 对账单 | 客户群收到用车记录后制作的对账凭证 | 客服绩效表、账单模板 | 财务、客服 | 是 |

## 需要统一核实的字段

- 客户编号前缀含义：无法确认。
- 订单号是否存在独立字段：无法确认。
- 客户归属和销售人是否同一口径：无法确认。
- “合计收款”“实际流水金额”“欠款金额”的最终口径：无法确认。
- 订单利润是否包含车损、油费、停车费、优惠、平台费、维修等：无法确认。
- 车辆状态枚举和订单状态枚举：无法确认。
