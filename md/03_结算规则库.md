## 自有车结算

| 项目 | 规则 |
|---|---|
| 触发条件 | 订单使用自有车或车辆台账标识为自有车。 |
| 计算逻辑 | 已发现销售统计中“租车成本”可为 0，利润字段为“最终销售金额 - 租车成本”；但自有车是否还扣保险、维修、折旧、洗车、停车等，无法确认。 |
| 例外情况 | 车辆维修、保险、车损、保养是否分摊到订单，无法确认。 |
| 证据来源 | `李楠.xlsx` 销售统计表；`自有车辆11月以前数据` 相关车辆文件；`车辆维修、整备管理相关表格（分报备、登记）.md`。 |
| 可信度 | 低到中：字段存在，但完整结算规则无法确认。 |

## 挂靠车结算

| 项目 | 规则 |
|---|---|
| 触发条件 | 车型/摘要出现“挂靠”，或支出付款类型出现“挂靠结算”。 |
| 计算逻辑 | 销售统计表中挂靠车有“租车成本”；利润字段可按“最终销售金额 - 租车成本”计算。挂靠车主具体分成比例或结算周期无法确认。 |
| 例外情况 | 挂靠群未回复、挂靠用车未结算被登记为违规，说明挂靠结算需要人工跟进。 |
| 证据来源 | `李楠.xlsx`；`绩效登记表(5).xlsx` / `业务部`；日收支明细中的“挂靠结算”。 |
| 可信度 | 中：有字段和异常记录，缺少制度性分成公式。 |

## 车队结算/绩效

| 项目 | 规则 |
|---|---|
| 触发条件 | 洗车、收车、送车、配驾。 |
| 计算逻辑 | 表内按日期、动作、客户编号、地点、是否异地、车型、车牌、今日收入登记，并用 `SUM` 合计。具体标准单价只在样例中出现，未形成制度性公式。 |
| 例外情况 | 异地、婚庆配驾、半天配驾、未收配驾费等特殊备注存在。 |
| 证据来源 | `施嘉祥2026年1月绩效(1).xlsx` / `Sheet1`。 |
| 可信度 | 中。 |
